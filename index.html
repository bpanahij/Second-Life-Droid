Welcome to Second Life Droid

This bundle of scripts functions together as an autonomous battle robot (droid) in Second Life. Your just have to put it together into a set of primitives and the scripts will seek out and begin to communicate with one another via messaging. Each script needs it's own prim and the weapons scripts will generate projectile prims, so they should probably go in som sort of gun-like prim. $ cd your_repo_root/repo_name $ git fetch origin $ git checkout gh-pages
